# classNames()

Implement the `classNames()` function, a commonly-used utility in modern front-end applications to conditionally join CSS class names together. This function is widely used in React applications and frameworks to manage dynamic class names.

## Function Signature

```ts
function classNames(...args: Array<ClassValue>): string
```

## Types

```ts
type ClassValue = ClassArray | ClassDictionary | string | number | null | boolean | undefined;
type ClassDictionary = Record<string, any>;
type ClassArray = Array<ClassValue>;
```

## Features

- ✅ Accepts strings, objects, arrays, and numbers
- ✅ Recursively flattens nested arrays
- ✅ Filters out falsey values (null, undefined, false, 0, '')
- ✅ Includes object keys with truthy values
- ✅ Returns space-separated string with no extra whitespace

## Examples

### Strings
```ts
classNames('foo', 'bar');                    // 'foo bar'
classNames('text', 'large', 'bold');         // 'text large bold'
```

### Objects
```ts
classNames({ 'foo-bar': true });             // 'foo-bar'
classNames({ 'foo-bar': false });            // ''
classNames({ foo: true, bar: false });       // 'foo'
classNames({ foo: true }, { bar: true });    // 'foo bar'
```

### Mixed
```ts
classNames('foo', { bar: true });            // 'foo bar'
classNames('foo', { bar: true, baz: false }, 'qux'); // 'foo bar qux'
```

### Arrays
```ts
classNames(['foo', 'bar']);                  // 'foo bar'
classNames('a', ['b', { c: true }]);         // 'a b c'
classNames(['a', ['b', ['c']]]);             // 'a b c' (nested arrays flattened)
```

### Falsey Values
```ts
classNames(null, false, 'bar', undefined);   // 'bar'
classNames('foo', false && 'bar', 0, '');    // 'foo'
```

## Use Cases

- **React Components**: Conditionally apply CSS classes based on props
- **Dynamic Styling**: Build class names based on component state
- **Theme Switching**: Toggle between theme classes
- **Responsive Design**: Apply classes based on breakpoints
- **Accessibility**: Add aria classes conditionally
